# задание

## Задача - вывод таблицы температур по Фаренгейту и перевод их по Цесльсию

Создать:

    - main.c — файл с кодом;

    - Makefile — исправить TARGET = c_2

Запустить сборку:

    make

Файл программы c_2.exe создатся в директории bin

Запустить программу:

    ./bin/c_2.exe

### Часть первая

В консоли должна появиться таблица из двух столбцов в таком виде:

    0   -17
    20  -6
    ...
    300  148

первый столбец Фаренгейт с шагом 20

втрой столбец перевод в Цельсия.

В Makefile исправить:

    SRC_FILES = main_1.c

### Часть вторая

Переделать первый вариант так, чтобы использовались вещественные числа с одной цифрой после десятичной точки
и выравниванием цифр по правому краю.

в консоли должна появиться таблица из двух столбцов в таком виде:

    0   -17.8
    20   -6.7
    ...

В Makefile исправить:

    SRC_FILES = main_2.c

### Часть третья

Задача - вывод таблицы температур по Цельсию и перевод их в Фаренгейт

Вывести заголовок таблицы

В Makefile исправить:

    SRC_FILES = main_3.c

### Часть четвёртая

Задача - использование цикла for()

Переделать программу, сократить кол-во операторов за счёт использования другой конструкции цикла.

В Makefile исправить:

    SRC_FILES = main_4.c
