# задание с_4

## Задача -  Символьный ввод / вывод

* Необходимо расчитать количество строк и количество символов введенных с клавиатуры
* Вывести результаты в терминал.

Создать:

    - main.c — файл с кодом;

    - Makefile — исправить TARGET = c_4

Запустить сборку:

    make

Исправить найденные ошибки в коде.
Файл программы c_4.exe будет создан в директории bin

Запустить программу:

    ./bin/c_4.exe

Проверить результат вывода и потом очистить проект в консоли VS Code:

    make clean

### Дополнительно

#### Программа main_1.c для подсчёта символов - пробелы, табуляция, конец строки

Задача - вывести таблицу с количеством каждого найденого символа

* EOL - конец строки
* TAB - табуляция
* SPACE - пробел

В Makefile исправить:

    SRC_FILES = main_1.c

---
Собрать и запустить:

    make
    ./bin/c_4.exe

В консоли должна появиться таблица из двух столбцов в таком виде:

    SYMBOL    TOTAL

    EOL       3
    TAB       0
    SPACE     4

---
Очистить проект в консоли VS Code:

    make clean

#### Программа main_2.c для подсчёта строк, слов и символов

Задача - вывести таблицу с количеством каждого из параметров

* LINES - строки
* WORDS - слова
* CHARS - символы

В Makefile исправить:

    SRC_FILES = main_2.c

---
Собрать и запустить:

    make
    ./bin/c_4.exe

В консоли должна появиться таблица из двух столбцов в таком виде:

    ---------
    LINES   3
    WORDS   0
    CHARS   4
    ---------

---
Очистить проект в консоли VS Code:

    make clean
