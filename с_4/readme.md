# задание с_4

## Задача -  Символьный ввод / вывод

* 1 Необходимо расчитать количество строк и количество символов введенных с клавиатуры
* 2 Вывести результаты в терминал.

Создать:

    - main.c — файл с кодом;

    - Makefile — исправить TARGET = c_4

Запустить сборку:

    make

Исправить найденные ошибки в коде.
Файл программы c_4.exe будет создан в директории bin

Запустить программу:

    ./bin/c_4.exe

Проверить результат вывода и потом очистить проект в консоли VS Code:

    make clean
