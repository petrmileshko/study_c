# задание с_4

## Задача -  Символьный ввод / вывод

* Необходимо расчитать количество строк и количество символов введенных с клавиатуры
* Вывести результаты в терминал.

Создать:

    - main.c — файл с кодом;

    - Makefile — исправить TARGET = c_4

Запустить сборку:

    make

Исправить найденные ошибки в коде.
Файл программы c_4.exe будет создан в директории bin

Запустить программу:

    ./bin/c_4.exe

Проверить результат вывода и потом очистить проект в консоли VS Code:

    make clean

### Дополнительно

#### Программа подсчёта символов - пробелы, табуляция, конец строки.

Задача - вывести таблицу с количеством каждого найденого символа

В Makefile исправить:

    SRC_FILES = main_2.c

---
Собрать и запустить:

    make
    ./bin/c_4.exe

В консоли должна появиться таблица из двух столбцов в таком виде:

    SYMBOL    TOTAL

    Enter     3
    Tab       1
    Space     4

---
Очистить проект в консоли VS Code:

    make clean
