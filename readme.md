# Язык Си

## Настройка среды для сборки

 1. Скачать и установить MSYS2 с [официального сайта](https://www.msys2.org/).
 2. Обновить пакетный менеджер и базовые пакеты. После установки запустить терминал MSYS2 и выполнить команду: pacman -Su
 3. Установить компилятор. В терминале MSYS2 выполнить команду: pacman -S mingw-w64-ucrt-x86_64-gcc
 4. Установить сборщик Make. В терминале MSYS2 выполнить команду: pacman -S make
 5. В переменную среды PATH добавить путь к компилятору gcc и сборщику make.
 6. Установить редактор VS Code, и расширения C/C++, C/C++ Extension Pack, CMake Tools, Code Runner, EditorConfig, Git History.
