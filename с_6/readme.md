# Задание с_6

## Задача: Подсчёт длины слов во входном потоке

* Завершение программы при нажатии Ctrl-X.
* Перед началом ввода, вывести инструкцию для пользователя.
* Слова разделяются пробелом, табуляцией, конецом строки.

Создать:

    - main.c — файл с кодом;

    - Makefile — исправить TARGET = c_6

Запустить сборку:

    make

Исправить найденные ошибки в коде.
Файл программы c_6.exe будет создан в директории bin

Запустить программу:

    ./bin/c_6.exe

Проверить результат вывода и потом очистить проект в консоли VS Code:

    make clean

В консоли должна появиться инструкция для ввода:

    Для завершения программы:
    1. Нажать комбинацию Ctrl + X
    2. Нажать клавишу ENTER
    Начните ввод текста >>>

### Вариант 1

После окончания ввода вывести таблицу данных в виде:

        Длина | Слов
            4 | 3
            5 | 1
            1 | 4
            2 | 2

Первый столбец длина слов, второй количество слов данной длины.

### Вариант 2

После окончания ввода вывести результаты в виде гистограммы:

        Длина  | Слов
        _____________
            1  | ****************************************
            4  | ******************************
            5  | ********************
            50 | **********
