# задание с_5

## Задача -  рефакторинг кода программы c_4/main_2.c

* Необходимо сделать завершение программы при нажатии Ctrl-X
* Вывод сообщений в терминале должен быть на кириллице.
* Перед началом ввода, вывести инструкцию для пользователя.

Создать:

    - main.c — файл с кодом;

    - Makefile — исправить TARGET = c_5

Запустить сборку:

    make

Исправить найденные ошибки в коде.
Файл программы c_5.exe будет создан в директории bin

Запустить программу:

    ./bin/c_5.exe

Проверить результат вывода и потом очистить проект в консоли VS Code:

    make clean

В консоли должна появиться инструкция и после окончания ввода таблица:

    Для завершения программы:
    Нажать комбинацию Ctrl + X

    Начните ввод текста >>>

    СЧЁТЧИК   ВСЕГО

    Строк      3
    Слов       0
    Символов   4

---
Очистить проект в консоли VS Code:

    make clean
