# задание с_1 Программа «Hello world!»

* Задача - вывод текста в консоль: Hello world!

создать:

    - main.c — файл с кодом;

    - Makefile — файл для сборки.
---
Запустить сборку в консоли VS Code:

    make

Файл программы c_1.exe создатся в директории bin

Запустить программу в консоли VS Code:

    ./bin/c_1.exe

в консоли должна появиться строка - Hello world!

---
Очистить проект в консоли VS Code:

    make clean
