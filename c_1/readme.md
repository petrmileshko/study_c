# задание

* Задача - вывод текста в консоль: "Hello world!"

создать:

    - main.c — файл с кодом;

    - Makefile — файл для сборки.

Запустить сборку:

    - команда в консоли make

Файл программы c_1.exe создатся в директории bin

Запустить программу:

   ./bin/c_1.exe

в консоли должна появиться строка - Hello world!
