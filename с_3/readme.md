# задание с_3 Рефакторинг кода программы с_2

## Задача - рефакторинг кода: оформить программу из [с_2 ч.6](https://github.com/petrmileshko/study_c/tree/main/с_2#часть-шестая) в соответсвии со стандартом C23

Создать:

    - main.c — файл с кодом согласно C23;

    - Makefile — исправить TARGET = c_3 и SRC_FILES = main.c

    - Makefile — проверка C23: добавить в CFLAGS флаги -std=c23 -pedantic -Werror

Запустить сборку:

    make

Исправить найденные ошибки в коде.
Файл программы c_3.exe будет создан в директории bin

Запустить программу:

    ./bin/c_3.exe

Проверить результат вывода и потом очистить проект в консоли VS Code:

    make clean
